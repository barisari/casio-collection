# Casio Collection

Kişisel Casio saat koleksiyonumu sergileyen bir web sayfası. Vintage/Retro, G-Shock ve Edifice serilerinden 18 saat içerir.

## Özellikler

- Light, Dark ve Retro tema desteği
- Grid ve liste görünümü
- Seri bazlı filtreleme ve arama
- Detaylı saat bilgisi modalı
- CSV export

## Geliştirme

Claude branch'lerine (`claude/*`) yapılan push'lar otomatik olarak `main` branch'ine merge edilir.
